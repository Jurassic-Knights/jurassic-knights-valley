# Workflows

When a task matches a workflow, @ the file or ask by name, then read and follow it.

| When to use | Workflow | Path |
|-------------|----------|------|
| New feature (plan only) | Feature Request | `feature.md` |
| New image/audio/asset | Asset Creation | `asset-creation.md` |
| Add procedural sound | Add Sound | `add-sound.md` |
| Regenerate / fix asset | Regenerate | `regenerate.md` |
| Regenerate sound | Sound Regenerate | `sound-regenerate.md` |
| Full codebase audit | System Audit | `system-audit.md` |
| Design / UX review | Design Audit | `design-audit.md` |
| Performance review | Performance Audit | `performance-audit.md` (and **performance-optimization** skill) |
| Code review by rubric | Gemini Review | `gemini-review.md` |
| New idea / exploration | Idea | `idea.md` |
| Cleanup / hygiene | Clean | `clean.md` |
