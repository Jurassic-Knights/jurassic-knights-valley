# SFX Asset Manifest

This list documents all required sound effects for *Jurassic Knights: Valley*.
Current status for all assets is **Pending Integration** (Placeholders).

## Hero
- `sfx_hero_shoot`: High-caliber rifle shot.
- `sfx_hero_swing`: Heavy tool swing (Whoosh).
- `sfx_hero_impact_flesh`: Bullet/Shovel hitting meat.
- `sfx_hero_impact_metal`: Shovel hitting rock/metal.
- `sfx_hero_step`: Heavy bootstep on dirt/mud.

## Resources
- `sfx_resource_break_wood`: Tree falling / wood splintering.
- `sfx_resource_break_stone`: Rock shattering.
- `sfx_resource_break_metal`: Metal clang/crunch.
- `sfx_resource_collect`: Pickup sound (coin/item).

## Dinosaurs
- `sfx_dino_roar`: Aggressive roar (Medium/Large).
- `sfx_dino_attack`: Bite/Snap sound.
- `sfx_dino_hurt`: Pain grunt.
- `sfx_dino_death`: Heavy body fall / death exhale.
- `sfx_dino_respawn`: Juicy supernatural/biology pop.

## UI
- `sfx_ui_click`: Mechanical button press.
- `sfx_ui_error`: Buzzer / "Access Denied".
- `sfx_ui_purchase`: Cash register / coins jingle.
- `sfx_ui_unlock`: Epic "Sector Unlocked" thud/chord.

## Game
- `sfx_level_up`: Positive fanfare (Short).
