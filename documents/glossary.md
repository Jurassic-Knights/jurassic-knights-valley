# Glossary

Master definitions for Jurassic Knights: Valley terminology.

## Core Terms

| Term | Definition |
|------|------------|
| **The Valley** | The game world; a prehistoric landscape where knights battle |
| **Knight** | Player character class; armored warriors |
| **Raptor** | Swift predatory dinosaur; common enemy/mount |
| **Forge** | Crafting/upgrade system |

## Resources

| ID | Display Name | Description |
|---|---|---|
| `gold` | Gold | Primary currency; earned by selling goods or finding caches. |
| `stamina` | Resolve | Physical/mental endurance; depleted by work, restored at Outpost (Rest) or by consuming items. |
| `xp` | Experience | Progression currency. |
| `scrap_metal` | Salvaged Scrap | Recovered metal from the ruined front; used for basic crafting. |
| `iron_ore` | Iron Deposit | Raw ore pulled from muddy trenches; refined into Ingots. |
| `fossil_fuel` | Black Tar | Combustible fuel (Coal/Oil) for war machines and Forges. |
| `wood` | Petrified Wood | Hardened ancient timber; fuel for Forges and basic construction. |
| `primal_meat` | Primal Meat | Raw flesh from prehistoric beasts (e.g., Velociraptors). |
| `item_scrap_plate` | Scrap Plate | Basic armor plating crafted from Scrap Metal. |
| `item_iron_ingot` | Iron Ingot | Refined iron, heavy and reliable. Crafted from Iron Ore. |
| `tool_shovel` | Trench Tool | Standard issue digging implement. Crafted from Scrap and Iron. |

## Gameplay Concepts

| ID | Display Name | Description |
|---|---|---|
| `sector` | Sector | Exploitable zone on the front lines; unlocked with Gold. |
| `home_outpost` | Home Outpost | Safe zone where troops recover Resolve (Stamina). |
| `forge` | Field Forge | Mobile crafting station; converts resources into goods (e.g., Ingots, Tools). |
| `infantry` | Infantry | Foot soldiers assigned to work or fight Sectors. |
| `primal` | Primal | Non-sapient dinosaur; provides resources when hunted. |
| `velociraptor` | Velociraptor | Fast predator. Drops: `Primal Meat`. |
| `tyrannosaurus` | Tyrannosaurus | Apex predator. Drops: `Iron Ore`. |
| `ankylosaurus` | Ankylosaurus | Armored herbivore. Drops: `Scrap Metal`. |
| `triceratops` | Triceratops | Horned herbivore. Drops: `Fossil Fuel`. |
| `parasaurolophus` | Parasaurolophus | Crested herbivore. Ambient/neutral. |
| `stegosaurus` | Stegosaurus | Plated herbivore. Ambient/neutral. |
| `spinosaurus` | Spinosaurus | Semi-aquatic predator. Rare spawn. |
| `pteranodon` | Pteranodon | Flying reptile. Ambient sky creature. |

## Terminology Guidelines

| Category | Words to Use | Words to AVOID |
|----------|--------------|----------------|
| **Structures** | Bunker, Citadel, Outpost, Trench | Dungeon, Castle (unless ruined), Spire |
| **Roles** | General, Scavenger, Quartermaster | Wizard, Rogue, NPC, Quest Giver |
| **Magic** | Willpower, Adrenaline, Tactics | Mana, Enchantment, Potion, Spell |
| **Actions** | Orders, Missions, Patrols | Quests, Raids (use 'Sorties') |

## Locations

| Name | Type | Description |
|------|------|-------------|
| **Mud Pots** | Shelter | Temporary dugouts for low-ranking survivors |
| **Stone Bunkers** | Fortification | Permanent defensive lines |
| **Capital Strongholds** | City | Heavily defended seats of High Command |


**Note**: This glossary is the source of truth. All agents should reference these terms for consistency.
